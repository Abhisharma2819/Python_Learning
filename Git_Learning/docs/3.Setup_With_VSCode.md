## 📌 Step 1: Install Git

1. Download Git from:
   👉 https://git-scm.com/download/win

2. Run the installer

3. Keep all settings as default (recommended)

4. Finish installation

---

## 🔍 Step 2: Verify Git Installation

Open **Command Prompt** or **VS Code Terminal** and run:

```bash id="gk3p8m"
git --version
```

### Example Output:

```text id="q8v2ld"
git version 2.50.1.windows.1
```

---

## ⚙️ Step 3: Configure Git Identity

Set your global Git username and email:

```bash id="r9k2zp"
git config --global user.name "Abhishek Sharma"
git config --global user.email "your-email@example.com"
```

### Check configuration:

```bash id="t3n8qv"
git config --list
```

---

### 📘 What this does:

These commands set your **global identity**, which will appear in all your commits.

---

## 🧰 Step 4: Install VS Code

Download VS Code:
👉 https://code.visualstudio.com

### Recommended Extensions:

* Python 🐍
* Pylance ⚡
* GitLens 🔍
* Jupyter 📊

---

## 🔗 Step 5: Connect VS Code with Git

Open VS Code → Open Terminal:

```
Terminal → New Terminal
```

Verify Git:

```bash id="m1p7xa"
git --version
```

If it returns a version number, Git is successfully connected.

---

## 📥 Step 6: Clone a Repository

Navigate to your working folder and run:

```bash id="c8v2lm"
git clone https://github.com/Abhisharma2819/Python_Learning.git
```

---

## 🔄 Daily Git Workflow

After making changes in your project:

```bash id="n9k3qp"
git status
git add .
git commit -m "Added pandas examples"
git push
```

---

## ⬇️ Pull Latest Changes

To get the latest updates from GitHub:

```bash id="x2m8zt"
git pull
```

---

## 🎯 Summary

| Command    | Purpose          |
| ---------- | ---------------- |
| git clone  | Copy repository  |
| git add .  | Stage changes    |
| git commit | Save changes     |
| git push   | Upload changes   |
| git pull   | Download updates |
